# Netflix Data Analysis 📊

## Overview

This project focuses on performing Exploratory Data Analysis (EDA) on a
Netflix dataset.\
The goal is to understand patterns, trends, and insights from the data
using Python.

------------------------------------------------------------------------

## Objectives

-   Analyze Netflix content data
-   Identify trends in movies and TV shows
-   Perform data cleaning and preprocessing
-   Generate meaningful insights using visualization

------------------------------------------------------------------------

## Tools & Technologies

-   Python
-   Pandas
-   NumPy
-   Matplotlib / Seaborn
-   Jupyter Notebook (Google Colab)

------------------------------------------------------------------------

## Key Steps Performed

1.  Data Loading and Inspection\
2.  Data Cleaning (handling null values, formatting)\
3.  Exploratory Data Analysis\
4.  Data Visualization\
5.  Insight Generation

------------------------------------------------------------------------

## Key Insights

-   Distribution of Movies vs TV Shows\
-   Trends over years\
-   Popular genres/categories\
-   Content growth patterns

------------------------------------------------------------------------

## Project Structure

    EDA-5.ipynb   -> Main analysis notebook
    README.md     -> Project documentation

------------------------------------------------------------------------

## How to Run

1.  Open the notebook in Google Colab or Jupyter Notebook\
2.  Run all cells step by step\
3.  Modify and explore the dataset as needed

------------------------------------------------------------------------

## Conclusion

This project helps in building a strong foundation in EDA and
understanding real-world datasets.\
It improves data analysis skills and visualization techniques.
